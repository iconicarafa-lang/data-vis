## Part 1: Track A - Social Network Visualization & Community Discovery

This track focuses on structural audit and narrative community discovery of the **Game of Thrones Character Interaction Network**. It applies Louvain community partitioning and betweenness centrality sizing to reveal character dynamics.

### Prerequisites

You need a Python environment with the following packages installed:
* `pandas`
* `networkx`
* `pyvis`
* `python-louvain`

### How to Run and View Track A

1. **Generate the Dataset**:
   Run the data preparation script to create the Node Table and Edge List in the `data/` folder:
   ```bash
   python src/data_prep.py
   ```

2. **Execute the Visualization and Analysis Pipeline**:
   Run the analysis script to apply Louvain partitioning, calculate centrality, and compile the interactive network:
   ```bash
   python src/visualize.py
   ```

3. **View the Visualization**:
   Open the generated interactive HTML file in any web browser:
   * Location: `dist/track_a_network.html`
   * Double-click the file to open it. You can zoom, hover over nodes for metrics, and drag nodes to inspect connections.

## Part 2: Track B - Real-Time Data Visualization Dashboard & Simulator

This track focuses on real-time price monitoring of volatile assets (Bitcoin & Ethereum) with dynamic sliding windows, threshold alerts, connection response latency metadata, and an interactive paper trading simulator.

### Prerequisites

In addition to Python, you need the following packages installed:
* `streamlit`
* `plotly`
* `requests`
* `pandas`

### How to Run and View Track B

1. **Run the Streamlit Dashboard**:
   From the project root directory, launch the Streamlit application:
   ```bash
   streamlit run src/dashboard.py
   ```

2. **View the Application**:
   Open the local URL generated by Streamlit (usually `http://localhost:8501`) in your web browser.

3. **Interactive Features**:
   * **Paper Trading**: Trade BTC and ETH using your $100M simulated cash balance.
   * **Win/Loss Scorecard**: Check the live card at the top to see if you are winning (gains) or losing (losses).
   * **Alerts**: Adjust price thresholds in the sidebar expander (`⚙️ Advanced Configuration`) and see card border transitions and flashing warning banners.
   * **Latency**: View live API request response speed (latency in milliseconds) on the third KPI card.
   * **Data Export**: Click `Export Session Price History` inside the second tab to download a CSV dataset of recorded ticks.
